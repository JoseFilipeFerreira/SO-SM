# SO-SM

Client-Server Stock Management (Operating Systems Assignment) 

## Setting up

```
cd src && make
```
* start server
```
./sv
```

* start product manager
```
./ma
```

* start client
```
./cv
```

More detailed information [here](https://github.com/JoseFilipeFerreira/SO-SM/blob/README/enunciado-so-2018-19.pdf).
